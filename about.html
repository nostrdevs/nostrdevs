About NostrDevs

NostrDevs is a community for those interested in discussing and participating in the research and development of Nostr and related protocols.

How It Works

This site is a curated collection of Nostr development resources designed specifically for meetup organizers and community members. Anyone can submit an item for inclusion by clicking the "Submit Item" button on the homepage, which creates a GitHub pull request.

Use the date filter to easily view everything new since your last meetup. Simply select the date of your last gathering, and you'll see all the resources, updates, and discussions that have been added since then. This makes it simple to stay current and prepare for your next community discussion.

You can also filter by tags such as "nips" (protocol specs), "clients" (applications), and "relays" (infrastructure) to focus on specific areas of interest.

Contributing

To contribute a new resource, click the "Submit Item" button on the homepage. You'll be guided through creating a GitHub pull request with your submission. All contributions are welcome!

Alternatively, you can manually add a JSON file to the /items directory in the GitHub repository.
